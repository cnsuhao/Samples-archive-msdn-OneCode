How to reload part of page content regularly (VBASPNETReloadPartContent)

Introduction

The VBASPNETReloadPartContent sample show you how to reload part of page content regularly.

​One method is use meta tag  to refresh page regually as show below:
HtmlMeta meta = new HtmlMeta();

meta.HttpEquiv = "Refresh";

meta.Content = "5";

this.Page.Header.Controls.Add(meta);

Here we want to use javascript setInterval with jquery ajax method.

Running the Sample

Please follow these demonstration steps below.

Step 1: Open the VBASPNETReloadPartContent.sln.

 

Step 2: Right-click the Default.aspx page then select "View in Browser".

 

Step 3: Validation finished.

Using the Code

Code Logical:                                                                                                                                                                                                                              

Step1. Create a VB "ASP.NET Web Application" in Visual Studio /Visual Web Developer. Name it as "VBASPNETReloadPartContent".

 

Step2. Add two Web Forms then rename them to ¡°Default.aspx¡± and ¡°Refresh.aspx¡±. The Default page is used to test part of page content regularly reload. The Refresh is used to get the latest data. Add an div to Default page, it is used to load the latest content. The main code will be shown as below:

- HTML code snippet -
 
<form id="form1" runat="server">
   This is the main page.
   <div>
       <div id="target">
       </div>
   </div>
   </form>
 
- end -


Add some Data Control to ¡°Refresh¡± page, they are used to bind data. In this demo, we add a Repeater Control to show data.

- HTML code snippet -
 
<asp:repeater id="rptResult" runat="Server">
           <HeaderTemplate>
<table>
           <tr>
               <th>
                   Id
               </th>
               <th>
                   Time
               </th>
           </tr>
</HeaderTemplate>
 
<ItemTemplate>
 <tr><td><%#Eval("Id")%></td><td><%# Eval("Time")%></td></tr>
</ItemTemplate> 
 
<FooterTemplate>
 </table>
</FooterTemplate>
      
   </asp:repeater>
 
- end -

Step3. Now we begin to implement reload part of page content regularly. Add JavaScript reference, and write the corresponding JavaScript script. In the Default page, the JavaScript will be shown as below:

- JavaScript code snippet -
 
<script src="jquery-1.4.1.min.js" type="text/javascript"></script>
 
    <script language="JavaScript" type="text/javascript">
        function refreshConsole() {
            $.ajax({
                type: "POST",
                url: "refresh.aspx",
                success: function(data) {
                    $("#target").html(data);
                }
            });
        }
 
        $(document).ready(function() {
            refreshConsole();
        });
        
    </script>
 
- end -

[Note] We need to set the value of the parameter type of jQuery Ajax method to "POST".

In the Refresh Page, the script as shown below:

- JavaScript code snippet -
 
<script type="text/javascript">
 
       setInterval(function() {
           load()
       }, 3000);
 
       var load = function() {
           location.reload();       
       };  
   </script>  
 
- end -

 

Step4. We get the latest Data and bind it to DataBind Control in the code-behind.

 

- VB code snippet -
 
Private Sub BindResult()
      'In the actual scene, you may not need operating the session, here is just to test
      If Session("dtResult") IsNot Nothing Then
          dtResult = DirectCast(Session("dtResult"), DataTable)
      Else
          dtResult = New DataTable()
          dtResult.Columns.Add("Id")
          dtResult.Columns.Add("Time")
      End If
 
      Dim dr As DataRow = dtResult.NewRow()
      dr("Id") = dtResult.Rows.Count + 1
      dr("Time") = DateTime.Now.ToString()
      dtResult.Rows.Add(dr)
 
      'Save data to Session, in a 
      Session("dtResult") = dtResult
      'Bind data to GridView
 
      rptResult.DataSource = dtResult
      rptResult.DataBind()
  End Sub
 
- end -

 

Step5. Build the application and you can debug it.
